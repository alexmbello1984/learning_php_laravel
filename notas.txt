crear archivo.htaccess acompa√±ando a index.php

<IfModule mod_rewrite.c> //si el modulo esta activo
	RewriteEngine on     //activar el motor 
	RewriteCond %{REQUEST_FILENAME} !-f  // siel archivo no existe
	RewriteRule ^(.*)$ index.php?url=$1 [QSA,L] // entonces ejecuta se envia a la ruta index.html?ruta+lo adicional

</IfModule>

de esta forma htacces se encargara de traducir una url amigable a la url como la necesita nuestro framework
